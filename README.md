# ThreeJSDemo

启动参考： https://v2.parceljs.cn/getting-started/webapp/

具体启动步骤：
使用 Yarn 来安装 Parcel：
`yarn add --dev parcel`
或者使用 npm 来安装 Parcel：
`npm install --save-dev parcel`

入口文件位置：src/index.html
运行 parcel
`yarn parcel src/index.html`
或者使用 npm 系列工具来运行：
`npx parcel src/index.html`

在浏览器中打开 http://localhost:1234/ 访问。
